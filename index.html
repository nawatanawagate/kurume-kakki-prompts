🔒G. 安全に使うガイド

お客様情報を守る・学習させない設定・商用利用の注意

学習させない設定（オプトアウト）
説明
📌 会話を学習に使われたくない
【Gemini】設定 →「Geminiアプリ アクティビティ」をオフ
【ChatGPT】設定 → データコントロール →「すべての人のためにモデルを改善する」をオフ
→ いずれも会話が学習に使われにくくなります（無料でも設定可）。
入れてはいけない情報
説明
📌 うっかり漏らさないために
【入力を避けるもの】
・お客様の名前／住所／電話／購入履歴
・仕入れ値や取引条件などの機密
→ 必要なときは「お客様A様」「○○市の方」のようにぼかして使う。
生成画像の商用利用の注意
説明
📌 販促に使う前に確認
【チェックポイント】
・他店やネットの写真を素材にしない（自分で撮った写真を使う）
・有名キャラ／ロゴ／実在人物の顔は避ける
・販促に使う前に各サービスの規約を確認
・店名／価格／日付などの事実は自分で記入し、人の目で最終チェック。
