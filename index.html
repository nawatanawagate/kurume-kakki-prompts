🔒I. 安全に使うガイド

お客様情報を守る・学習させない設定・商用利用・確認のルール

学習させない設定（オプトアウト）
説明
📌 会話を学習に使われたくない
【Gemini】設定 →「Geminiアプリ アクティビティ」をオフ
【ChatGPT】設定 → データコントロール →「すべての人のためにモデルを改善する」をオフ
→ いずれも会話が学習に使われにくくなります（無料でも設定可）。
入れてはいけない情報
説明
📌 うっかり漏らさないために
【入力を避けるもの】
・お客様の名前／住所／電話／購入履歴
・仕入れ値や取引条件などの機密
→ 必要なときは「お客様A様」「○○市の方」のようにぼかして使う。
生成画像の商用利用の注意
説明
📌 販促に使う前に確認
【チェックポイント】
・他店やネットの写真を素材にしない（自分で撮った写真を使う）
・有名キャラ／ロゴ／実在人物の顔は避ける
・販促に使う前に各サービスの規約を確認
・店名／価格／日付などの事実は自分で記入し、人の目で最終チェック。
著作権・写真の使い方
説明
📌 素材選びで迷う
【基本ルール】
・ネットで拾った写真やイラストをそのまま素材にしない
・お店で撮った写真、商用OKの素材、AIで作った画像を使う
・人物が写る写真は本人の許可を取る。
AIの答えの確認（ハルシネーション）
説明
📌 まちがいを防ぎたい
【気をつけること】
・AIは“もっともらしい間違い”を言うことがある
・育て方・効能・日付・価格などの事実は、必ず人が確認
・お客様に渡す前にもう一度読み直す。
お客様への伝え方
説明
📌 誤解を生まないために
【ひとこと添える】
・AIで作った画像は「イメージです」と添える
・実物と色や形が違う場合があることを伝える
→ 後のトラブルを防げます。
